me_va_qp.exe is the C program to deal with the mean-variance problem and calculate the portfolio.

test.txt is the test data containing the the covariance matrix Q, the mean asset return vector mu, the upper and lower bounds u_j and l_j of 100 stocks.

To run the program, open the terminal or cmd window and go to this folder, type "me_va_qp.exe test.txt" and then press ENTER.
